## How to Use This File

**At session start:**
1. Read this file to understand current state
2. Check git log for recent commits
3. If resuming, pick up from last `in_progress` step

**During work:**
1. Update step status as you progress
2. Work on ONE step at a time
3. Commit after completing each step

**At session end:**
1. Update step statuses
2. Add session log entry
3. Note any blockers
4. Describe next step clearly for handoff

**When task is complete:**
1. Mark all steps completed
2. Set status to `completed`
3. Clear steps for next task
